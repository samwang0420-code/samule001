# GEO + SEO 实施方法论

## 核心理念：双维度优化引擎

传统SEO关注搜索引擎排名，GEO关注AI平台引用。我们的方法论将两者融合，形成**双维度优化引擎**。

---

## 一、双维度分析框架

### 1.1 SEO维度（搜索引擎优化）

```
技术SEO (30%)
├── 网站速度优化
├── 移动适配
├── Schema标记
├── 索引覆盖率
└── Core Web Vitals

内容SEO (40%)
├── 关键词策略
├── 内容质量评分
├── E-E-A-T信号
├── 内容新鲜度
└── 语义相关性

权威SEO (30%)
├── 外链质量
├── 品牌提及
├── 本地引用(NAP)
├── 社交媒体信号
└── 行业权威度
```

### 1.2 GEO维度（生成引擎优化）

```
AI可见性 (50%)
├── Perplexity引用
├── ChatGPT知识
├── Google SGE引用
├── Bing Copilot引用
└── 其他AI平台

知识图谱 (30%)
├── Google Knowledge Panel
├── Bing Entity Graph
├── 结构化数据
├── 实体关联度
└── 品牌实体化

可信度信号 (20%)
├── 专家背书
├── 权威机构引用
├── 用户评价
├── 案例研究
└── 数据可信度
```

## 二、实施流程：5阶段闭环

### Phase 1: 双维度诊断（Day 1-2）

#### SEO诊断清单
- [ ] 技术SEO审计（工具：Screaming Frog, GTmetrix）
- [ ] 关键词现状分析（排名、流量、竞争度）
- [ ] 内容质量评估（原创度、深度、更新频率）
- [ ] 外链分析（数量、质量、锚文本分布）
- [ ] 本地SEO审计（GMB、 citations、reviews）

#### GEO诊断清单
- [ ] AI平台品牌提及扫描（Perplexity, ChatGPT）
- [ ] 知识图谱存在性检查
- [ ] 结构化数据完整性
- [ ] 实体关联度分析
- [ ] AI引用质量评估

**输出物**: 《双维度诊断报告》

---

### Phase 2: 策略制定（Day 3-5）

#### 2.1 关键词双维度矩阵

| 关键词 | 搜索量 | SEO难度 | GEO潜力 | 优先级 | 策略 |
|--------|--------|---------|---------|--------|------|
| 主词A | 高 | 中 | 高 | P0 | 双维度重点 |
| 长尾B | 中 | 低 | 中 | P1 | SEO主攻 |
| 问题C | 中 | 低 | 高 | P1 | GEO主攻 |

#### 2.2 竞品逆向工程

**SEO竞品分析**:
- 排名TOP3页面结构
- 内容长度、格式、更新频率
- 外链来源和质量
- 技术SEO优势

**GEO竞品分析**:
- AI平台引用来源
- 知识图谱实体关联
- 结构化数据类型
- 专家背书策略

#### 2.3 双维度内容策略

**内容类型矩阵**:
```
高搜索 + 高GEO价值 →  pillar content (支柱内容)
高搜索 + 低GEO价值 →  SEO-optimized pages
低搜索 + 高GEO价值 →  AI-friendly content
低搜索 + 低GEO价值 →  暂不处理
```

**输出物**: 《双维度优化策略书》

---

### Phase 3: 优化执行（Day 6-20）

#### 3.1 技术优化（Week 1）

**SEO技术栈**:
```
1. Schema标记部署
   ├── LocalBusiness (医疗诊所)
   ├── MedicalBusiness (医疗服务)
   ├── FAQPage (常见问题)
   ├── HowTo (操作指南)
   └── Review (评价聚合)

2. 页面速度优化
   ├── 图片压缩/懒加载
   ├── CDN部署
   ├── 代码分割
   └── 缓存策略

3. 索引优化
   ├── Sitemap更新
   ├── Robots优化
   ├── 内链结构
   └── 404修复
```

**GEO技术栈**:
```
1. 结构化数据增强
   ├── 实体标记 (Person, Organization)
   ├── 关系标记 (employee, founder)
   ├── 资质标记 (credential, award)
   └── 位置标记 (geo-coordinates)

2. 知识图谱优化
   ├── Wikipedia页面 (如可能)
   ├── Wikidata条目
   ├── Google Business优化
   └── 行业目录提交
```

#### 3.2 内容优化（Week 2-3）

**SEO内容模板**:
```
标题: 主关键词 + 地域词 + 修饰词 (60字符)
H1: 主关键词自然变体
H2: 相关长尾词 (3-5个)
内容结构:
  - 引言: 150字,包含主关键词
  - 正文: 1500-2500字
  - FAQ: 5-8个相关问题
  - CTA: 明确的行动号召
```

**GEO内容模板**:
```
AI友好结构:
  - 直接回答 (30-50字)
  - 详细解释 (200-300字)
  - 权威引用 (数据/研究)
  - 对比表格 (如适用)
  - 专家观点 (E-E-A-T)

引用优化:
  - 每段包含可引用语句
  - 关键数据突出显示
  - 权威来源标注
```

#### 3.3 权威建设（持续）

**SEO权威建设**:
- 高质量外链获取 (DA>50)
- 行业媒体投稿
- 专家采访/客座文章
- 本地引用建设

**GEO权威建设**:
- 行业报告引用
- 学术文献关联
- 权威机构背书
- 媒体提及监控

**输出物**: 《优化执行清单》+ 部署文件包

---

### Phase 4: 双维度监控（持续）

#### 4.1 SEO监控体系

**每日监控**:
```
- 核心关键词排名 (TOP20)
- 网站流量变化
- 索引状态检查
- 技术问题警报
```

**每周监控**:
```
- 排名变化趋势
- 竞品动态跟踪
- 外链新增/丢失
- 内容表现分析
```

**每月监控**:
```
- 全面SEO审计
- 策略效果评估
- 下月计划调整
```

#### 4.2 GEO监控体系

**AI平台监控**:
```bash
# Perplexity引用监控
- 品牌提及频率
- 引用来源分布
- 引用准确性评分

# ChatGPT知识监控
- 品牌知识准确性
- 知识更新时间
- 相关实体关联

# SGE引用监控
- 搜索结果引用
- 引用页面质量
- 引用位置分析
```

**知识图谱监控**:
- Google Knowledge Panel更新
- 实体关联变化
- 结构化数据效果

**输出物**: 《双维度监控仪表盘》

---

### Phase 5: 效果评估与迭代（Monthly）

#### 5.1 双维度KPI

**SEO KPI**:
- 关键词排名提升率
- 有机流量增长率
- 转化率优化
- 页面体验分数

**GEO KPI**:
- AI平台引用率
- 引用准确性
- 品牌提及情感
- 知识图谱完整度

**综合KPI**:
- 双维度ROI
- 客户获取成本
- 品牌认知度提升

#### 5.2 迭代优化

基于数据反馈:
1. 识别低效内容 → 优化或删除
2. 发现新机会 → 快速测试
3. 算法更新 → 策略调整
4. 竞品动作 → 应对策略
