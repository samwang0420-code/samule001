# GEO Client Onboarding System

## 客户签约后自动触发的工作流

### Phase 0: 数据收集（Day 1）
- [ ] 客户填写：律所名称、地址、主攻案件类型、目标城市
- [ ] 自动抓取：现有GMB数据、网站技术审计、竞品基准
- [ ] Apify任务队列启动：目标关键词SERP监控

### Phase 1: 毫米级地理审计（Day 2-3）
- [ ] 精确坐标标定（建筑入口级别）
- [ ] 周边50米POI地图（餐厅、停车场、法院、交通站）
- [ ] 停车便利性评分算法
- [ ] 地理精度竞品对比矩阵

### Phase 2: 技术基建（Day 4-7）
- [ ] Schema标记重构（LegalService + Attorney + LocalBusiness）
- [ ] 动态知识图谱初始化
- [ ] GMB优化（类别、属性、照片、Q&A）
- [ ] 本地引用（Citation）清洗与分发

### Phase 3: 内容注入（Week 2）
- [ ] 地理位置页面（Location Page）生成
- [ ] 律师个人资料页优化
- [ ] 案件类型→地理位置语义关联

### Phase 4: 监控部署（Week 2-3）
- [ ] 排名追踪看板（Grafana）
- [ ] 竞品波动监控
- [ ] 知识图谱变化检测
