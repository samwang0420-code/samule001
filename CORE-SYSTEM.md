# CORE SYSTEM - 护城河唯一目标

## 原则
**只有一个目标：能自动交付GEO结果的系统。**

砍掉：
- 网站前端
- 用户界面
- 支付集成
- 邮件服务

保留：
- 数据引擎
- 算法核心
- 自动化工作流
- API接口

## 核心交付流程

```
输入：律所名称 + 地址
    ↓
[Apify] 抓取GMB数据
    ↓
[算法] 计算GEO评分
    ↓
[生成器] 输出优化Schema
    ↓
[部署] 推送到客户端
    ↓
[监控] 追踪排名变化
    ↓
输出：排名提升报告
```
